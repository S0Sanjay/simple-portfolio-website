# Personal Portfolio Website

This is a simple personal portfolio website built using HTML and basic CSS. 
The website consists of two main pages: Home and Contact, designed with a clean and minimal layout.

## Pages
- Home – Displays personal information and an overview
- Contact – Provides contact details to reach me

## Features
- Clean and minimal design
- Built using HTML5 and CSS3 only
- Simple navigation between pages
- Easy to customize

## Technologies Used
- HTML5
- CSS3
